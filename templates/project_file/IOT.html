The Industrial Temperature Monitoring System is a simple system designed to measure and monitor temperature in industrial environments. It continuously checks the temperature and alerts the user when it exceeds a predefined limit, helping to prevent overheating and possible damage. The system is built using an Arduino Uno and a K-type thermocouple (MAX6675) for accurate temperature sensing. Temperature readings and the set threshold are displayed on a 16Ã—2 LCD using an I2C module. A 4Ã—4 keypad allows the user to easily set or modify the temperature limit as required.

When the temperature goes beyond the defined threshold, a buzzer provides an immediate alert, and a GSM800L module sends an SMS or makes a call to notify the user remotely. The project is low-cost, easy to use, and suitable for basic industrial safety applications, with future scope for features like data logging or mobile app connectivity.
